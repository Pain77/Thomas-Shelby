## ✨ Highlights.

-   Fully Modular Design
-   Object Oriented
-   Written in [TypeScript](https://www.typescriptlang.org/)
-   Self-Restoring Auth
-   Built with [Baileys](https://github.com/adiwajshing/baileys) (The Best
    WhatsApp Library Out There)
